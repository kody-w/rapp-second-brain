# 🧠 rapp-second-brain — the ecosystem that knows itself

A living map of the entire RAPP ecosystem — every repo, protocol, surface,
and relationship — built so that any person or any AI can orient in minutes
instead of months. This public repository is the **public hemisphere**;
a private hemisphere covers what the public one deliberately doesn't, and
the boundary between them is architectural: the public brain is built only
from public sources, ever.

Built on the shoulders of [kody-w/rapp-map](https://github.com/kody-w/rapp-map)
(the estate map and neuron mesh) — this is its deepening: two hemispheres,
a crawl discipline, and a freshness loop.

- **The mission**: [`OPUS-HANDOFF.md`](./OPUS-HANDOFF.md) — the full build
  plan, written as a baton.
- **The card format**: [`SCHEMA.md`](./SCHEMA.md)
- **The brain itself**: grows in `brain/` as the crawl proceeds.
